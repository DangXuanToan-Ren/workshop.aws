### Week 7 Objectives:
- Learn basic knowledge about Docker virtualization.
- Learn about Amazon ECR (Elastic Container Registry) image storage service.
- Practice backend development (e.g., Node.js/Express), write Dockerfile to package application into Container.
- Practice logging into ECR via CLI, build and push Docker image to ECR repository.
- Research Amazon ECS (Elastic Container Service) orchestration service and Task Definitions concept.
- Practice configuring ECS Cluster, launching Container on Serverless ECS Fargate platform.
