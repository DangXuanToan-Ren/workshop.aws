### Mục tiêu tuần 7:
- Học kiến thức cơ bản về ảo hóa Docker.
- Tìm hiểu dịch vụ lưu trữ image Amazon ECR (Elastic Container Registry).
- Thực hành lập trình backend (ví dụ: Node.js/Express), viết Dockerfile đóng gói ứng dụng thành Container.
- Thực hành đăng nhập ECR qua CLI, build và push Docker image lên kho lưu trữ ECR.
- Nghiên cứu dịch vụ điều phối Amazon ECS (Elastic Container Service) và khái niệm Task Definitions.
- Thực hành cấu hình ECS Cluster, khởi chạy Container trên nền tảng Serverless ECS Fargate.

- Học được kiến thức cơ bản về Docker và hiểu dịch vụ Amazon ECR.
- Lập trình backend thành công, viết Dockerfile đóng gói ứng dụng thành Container.
- Đăng nhập ECR qua CLI, build và push Docker image lên ECR thành công.

- Cấu hình thành công ECS Cluster và khởi chạy Container trên Serverless ECS Fargate.
