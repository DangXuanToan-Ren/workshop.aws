### Mục tiêu tuần 6:
- Nghiên cứu tổng quan bộ công cụ AWS Developer Tools.
- Tìm hiểu cách hoạt động của AWS CodeCommit và CodeBuild.
- Nghiên cứu luồng vận hành của AWS CodeDeploy và cấu trúc tự động hóa của CodePipeline.
- Chuẩn bị mã nguồn ứng dụng web nhỏ, thiết lập kết nối Git repository với AWS CodePipeline.
- Thực hành thiết lập luồng CI/CD tự động, viết cấu hình tự động build và deploy lên EC2.

### Kết quả đạt được tuần 6:
- Nghiên cứu được tổng quan bộ công cụ AWS Developer Tools, hiểu cách hoạt động của CodeCommit và CodeBuild.
- Hiểu được luồng vận hành của CodeDeploy và cấu trúc tự động hóa của CodePipeline.
- Chuẩn bị xong mã nguồn ứng dụng web và thiết lập kết nối Git repo với CodePipeline.
- Thiết lập thành công luồng CI/CD tự động và cấu hình tự động build khi có thay đổi code.
- Push code lên Git kích hoạt pipeline, theo dõi và deploy thành công bản cập nhật lên EC2.
