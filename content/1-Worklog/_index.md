This page will introduce my worklog, completing the program in 10 weeks (with Weeks 11 and 12 reserved for project finalization). The weekly contents are as follows:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)
- Getting to know team members, unit regulations
- Learning AWS, creating Free Tier account, Console and CLI
- Basic EC2 practice

**Week 2:** [Learning to work with AWS Management Console and CLI](1.2-week2/)
- Studying synchronization between Console and CLI
- Practicing Key Pairs and Security Groups management
- Synthesizing connection and resource management skills

**Week 3:** [Analyzing Compute, Storage, Networking and Database service groups](1.3-week3/)
- In-depth study of service groups
- Practicing checking and operating via Console and CLI
- Writing summary logs and evaluations

**Week 4:** [Enhancing EC2 and EBS management skills](1.4-week4/)
- Reinforcing EC2, AMI, EBS knowledge
- Practicing creating and attaching Elastic IP
- Managing EC2 lifecycle via CLI
- Configuring backup with EBS Snapshot

**Week 5:** [Building Serverless systems with Lambda and Event-Driven](1.5-week5/)
- Lambda programming, integrating Slack notifications
- Practicing CloudWatch and Grafana
- Tag-based resource management, IAM authorization

**Week 6:** [Building CI/CD with AWS Developer Tools](1.6-week6/)
- Studying CodeCommit, CodeBuild, CodeDeploy, CodePipeline
- Preparing source code, setting up Git connection
- Practicing setting up automatic CI/CD pipelines

**Week 7:** [Containerization and deployment with ECR and ECS](1.7-week7/)
- Learning Docker, ECR, ECS
- Writing Dockerfile, building and pushing image to ECR
- Configuring ECS Cluster, running container on Fargate

**Week 8:** [Building RESTful API with API Gateway and DynamoDB](1.8-week8/)
- Creating REST API with API Gateway
- Storing data in DynamoDB
- Practicing CRUD via API

**Week 9:** [Integrating authentication and optimizing Frontend distribution](1.9-week9/)
- Studying Amazon Cognito
- Integrating authentication with API Gateway
- Hosting Frontend on S3, configuring CloudFront

**Week 10:** [Automation with Infrastructure as Code (AWS SAM)](1.10-week10/)
- Learning Infrastructure as Code (IaC)
- Writing AWS SAM templates
- Deploying entire serverless stack

**Week 11:** [Setting up CI/CD pipeline for Frontend](1.11-week11/)
- Researching CI/CD pipeline for Frontend applications
- Setting up Git repository and automatic build
- Deploying static files to S3 and configuring CloudFront invalidation
- Testing the complete CI/CD workflow

**Week 12:** Not updated yet
