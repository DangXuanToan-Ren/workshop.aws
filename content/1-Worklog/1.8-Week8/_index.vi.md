### Mục tiêu tuần 8:
- Nghiên cứu khái niệm kiến trúc Serverless trên nền tảng AWS và các lợi ích tối ưu chi phí.
- Thực hành cấu hình và khởi tạo các endpoint RESTful API thông qua dịch vụ Amazon API Gateway.
- Thực hành viết code xử lý logic nghiệp vụ cho API (ví dụ bằng Node.js) trên môi trường AWS Lambda.
- Tìm hiểu cơ sở dữ liệu NoSQL Amazon DynamoDB, thực hành tạo bảng (Table) trên DynamoDB.
- Hoàn thiện ứng dụng Serverless: Tích hợp API Gateway gọi Lambda, cho phép Lambda lưu/truy xuất dữ liệu trực tiếp với DynamoDB.

### Kết quả đạt được tuần 8:
- Nghiên cứu được khái niệm Serverless trên AWS và hiểu các lợi ích tối ưu chi phí.
- Cấu hình và khởi tạo thành công các endpoint RESTful API bằng Amazon API Gateway.
- Viết code xử lý logic nghiệp vụ thành công trên AWS Lambda.
- Tìm hiểu được DynamoDB và tạo được bảng (Table) trên DynamoDB.
- Hoàn thiện thành công ứng dụng Serverless: Tích hợp API Gateway, Lambda và DynamoDB để lưu/truy xuất dữ liệu.
